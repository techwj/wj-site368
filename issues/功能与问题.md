
## 1.网站功能概述与布局更新

### 1.概述
> 基于当前网站的基本架构(一个页面+worker.js) + wiki提供的历史上今天的api, 来获取相关信息展示在页面上.

api文档: https://api.wikimedia.org/wiki/Feed_API/Reference/On_this_day#JavaScript
api: https://api.wikimedia.org/feed/v1/wikipedia/en/onthisday/all/${month}/${day}
api参数:
Parameters
language
required path

Language code. For example: ar (Arabic), en (English), es (Spanish). List supported languages.
type
required path

Type of event:
all: Returns all types
selected: Curated set of events that occurred on the given date
births: Notable people born on the given date
deaths: Notable people who died on the given date
holidays: Fixed holidays celebrated on the given date
events: Events that occurred on the given date that are not included in another type
MM
required path

Zero-padded month, 01 through 12
DD
required path

Zero-padded day of the month, 01 through 31

## 3. 多语言实现方案

### 3.1 技术架构
多语言功能基于前端JavaScript实现，采用JSON文件存储翻译内容的方式：

**核心文件结构：**
- `public/i18n.js` - 多语言核心类和逻辑
- `public/i18n/{lang}.json` - 各语言翻译文件（如：zh.json, en.json, de.json等）

### 3.2 实现机制

#### 3.2.1 语言检测与切换
1. **检测优先级：**
   - URL参数 `?lang=en` 优先级最高
   - localStorage保存的用户偏好
   - 浏览器语言设置
   - 默认英语fallback

2. **语言持久化：**
   - 用户选择的语言保存到localStorage
   - URL参数自动更新
   - 页面刷新后保持选择

#### 3.2.2 翻译文件结构
每个语言的JSON文件包含以下结构：
```json
{
  "meta": {
    "title": "页面标题",
    "description": "页面描述",
    "keywords": "关键词"
  },
  "nav": {
    "today": "导航项",
    "events": "事件"
  },
  "dateSelector": {
    "title": "选择日期",
    "month": "月份",
    "day": "日期"
  },
  "results": {
    "births": "出生的名人",
    "events": "历史事件"
  },
  "reviews": {
    "title": "用户评价",
    "users": [...]
  },
  "faq": {
    "title": "常见问题",
    "items": [...]
  }
}
```

#### 3.2.3 页面元素翻译
1. **HTML属性标记：**
   - `data-i18n="key.path"` - 翻译文本内容
   - `data-i18n-placeholder="key"` - 翻译input placeholder
   - `data-i18n-aria="key"` - 翻译aria-label属性

2. **动态内容翻译：**
   - SEO meta标签自动更新
   - FAQ问答内容动态替换
   - 用户评论内容本地化
   - 模态框内容翻译

#### 3.2.4 SEO优化
1. **hreflang标签：** 每种语言都有对应的hreflang链接
2. **动态meta更新：** title、description、og标签根据语言动态更新
3. **URL参数：** 通过?lang=xx参数支持搜索引擎索引

### 3.3 支持的语言列表
当前支持12种语言，与Wikipedia API支持的语言保持一致：
- en (English) - 英语
- zh (中文) - 中文
- de (Deutsch) - 德语
- fr (Français) - 法语
- it (Italiano) - 意大利语
- es (Español) - 西班牙语
- tr (Türkçe) - 土耳其语
- ar (العربية) - 阿拉伯语
- bs (Bosanski) - 波斯尼亚语
- uk (українська) - 乌克兰语
- sv (Svenska) - 瑞典语
- pt (Português) - 葡萄牙语

### 3.4 实现特点
1. **轻量级：** 纯前端实现，无需服务端支持
2. **响应式：** 支持实时语言切换，无需刷新页面
3. **兼容性：** 向后兼容，fallback到英语
4. **可扩展：** 易于添加新语言支持
5. **SEO友好：** 搜索引擎可索引多语言版本
